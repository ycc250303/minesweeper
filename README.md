# 扫雷小游戏
同济大学2025破壁工作室暑期入门考核：基于微信小程序的扫雷游戏。

## 功能特性

- 三种预设难度：初级(9×9, 10雷)、中级(12×12, 20雷)、高级(16×16, 40雷)
- **自定义模式**：可以自定义行数(9-16)、列数(9-16)、雷数(10-40)
- 支持点击和长按操作
- 实时计时和旗子计数
- 游戏结果提示

## 自定义模式使用说明

1. 在首页点击"自定义模式"按钮
2. 在弹出的输入面板中输入：
   - 行数：9-16之间的数字
   - 列数：9-16之间的数字  
   - 雷数：10-40之间的数字
3. 系统会自动验证输入的有效性：
   - 确保所有输入都是有效数字
   - 确保行列数在指定范围内
   - 确保雷数不超过总格子数的30%
4. 点击"确认"开始游戏，或点击"取消"关闭面板

## 游戏操作

- **单击**：翻开格子
- **长按**：插旗/取消插旗
- **双击**：快速翻开周围格子（当周围旗子数量等于周围雷数时）

## 技术栈

- 微信小程序
- TypeScript
- WXML + WXSS

## 开发环境

确保已安装微信开发者工具，并导入项目即可开始开发。
